# Tour Proyek Perpustakaan Laboratorium Pemrograman

### Install Project (Wajib dijalankan)
```
git clone :via https/ssh
cd project
composer i
cp .env.example .env // FILESYSTEM_DISK=public
php artisan key:generate
npm i
npm run build
npm run dev
php artisan migrate
php artisan db:seed
php artisan serve
```

### Feature
1. File Upload
2. Livewire Form
3. Login System by Breeze
